Langkah awal membuka SIMETRISS

### Login

[https://simetriss.rsupprofngoerah.my.id/main/login](https://simetriss.rsupprofngoerah.my.id/main/login)

Masukan username pada kolom username, jika user belum pernah sama sekali login SIMETRISS masukan username SIMRS senulumnya. Dan password menggunakan **@RsNgoerah1** jika user sama sekali belum pernah login SIMETRISS

Jika berhasil login, maka tampil halaman **Home** seperti di bawah.
