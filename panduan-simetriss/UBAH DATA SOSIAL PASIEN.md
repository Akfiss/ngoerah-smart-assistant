Digunakan untuk melakukan perubahan data sosial pasien jika terjadi kesalahan dalam pengisian data demografi

### Cari Data Pasien

Lakukan pencarian data pasien dapat dilakukan dengan melakukan **klik** _tombol cari data pasien_ atau langsung memasukan _nomor rekam medis di isian NO. RM_ dan kemudian tekan “**enter**”  untuk menampilkan biodata pasien

### Ubah

Setelah data pasien muncul. Tekan tombol “**ubah”** untuk mengaktifkan kolom isian yang akan dirubah atau ditambahkan

### Isi Data Perubahan

Lakukan perubahan data demografi sesuai kebutuhan.

### Simpan Perubahan

Tekan tombol “**simpan”** jika sudah selesai melakukan perubahan data

1.  Kesalahan input **Data Lain » Data Pendidikan/Pekerjaan Pasien** pada Surat Keterangan Sehat, maka untuk melakukan perubahan data tersebut bisa dilakukan pada menu **Ubah Data Sosial Pasien** ini.
    
2.  Kesalahan input Nama **Orang Tua** ketika _Registrasi Bayi_ di Menu **Pelayanan** berpengaruh pada Surat Keterangan Lahir. Maka, lakukan perubahan data sosial dengan menginput No.RM bayi
